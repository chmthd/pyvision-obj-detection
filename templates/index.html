Application Objective

PyVision is a solution designed to leverage real-time object detection using the YOLOv8 model. Built with FastAPI for the backend, it seamlessly integrates and deploys both API and frontend files. It aims to provide an easy-to-use interface for detecting objects using your camera.

Advanced Features

It displays a live video feed from your camera upon permission. It allows you to capture images and visualize detected objects with bounding boxes. PyVision's responsive design ensures a consistent experience across various handheld devices. The automated deployment and dynamic ngrok URL handling simplify development and remote access.

Setup and Deployment

This application uses FastAPI for a modern, high-performance backend and serves the frontend with HTML, CSS, and JavaScript. Implementing the YOLOv5 model pretrained on the COCO dataset, it offers real-time object detection. Deployment is streamlined with ngrok, creating secure tunnels to localhost and making the application accessible over the internet.
